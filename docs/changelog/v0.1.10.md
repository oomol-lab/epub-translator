# Release v0.1.10

This release improves compatibility with OpenAI-compatible providers and fixes GPT-5 request handling.

## ✨ Improvements

### Added `extra_body` support to `LLM` (#130)

`LLM` now supports an optional `extra_body` parameter, so provider-specific OpenAI-compatible request options can be passed through when needed.

Related PR: https://github.com/oomol-lab/epub-translator/pull/130
Related Issue: https://github.com/oomol-lab/epub-translator/issues/129

## 🐛 Bug Fixes

### Fixed GPT-5 request errors caused by `null` optional params (#136)

Optional request parameters such as `max_tokens`, `temperature`, and `top_p` are now omitted when their value is `None`, avoiding GPT-5 API errors caused by sending `null`.

Related PR: https://github.com/oomol-lab/epub-translator/pull/136
Related Issue: https://github.com/oomol-lab/epub-translator/issues/135
